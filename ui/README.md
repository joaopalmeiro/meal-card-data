# mealcard

Places that accept or not the Edenred meal card.

## Development

Install [fnm](https://github.com/Schniz/fnm) (if necessary).

```bash
fnm install && fnm use && node --version && npm --version
```

```bash
npm install
```

```bash
npm run dev
```

```bash
npm run lint
```

```bash
npm run format
```

```bash
npm run preview
```

```bash
npm run build
```
